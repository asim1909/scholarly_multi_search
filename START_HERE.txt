🎯 HOW TO USE - THREE SIMPLE OPTIONS
═════════════════════════════════════════════════════════════════════════════════

OPTION 1: Interactive Menu (Easiest) ⭐
─────────────────────────────────────
  python quickstart.py
  └─ Follow on-screen menu
  └─ Great for first-time users

OPTION 2: Command Line (Recommended)
──────────────────────────────────────
  # Search for papers
  python -m scholarly.cli search -q "machine learning" -n 50 -o papers.xlsx
  
  # Download papers
  python download_papers.py papers.xlsx
  
  # All papers saved to papers/ folder ✓

OPTION 3: Direct Download
──────────────────────────
  python download_papers.py existing_file.xlsx --dir my_papers --max 20
